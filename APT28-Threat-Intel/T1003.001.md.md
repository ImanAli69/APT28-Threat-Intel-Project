# T1003.001

## Technique Name
OS Credential Dumping: LSASS Memory

## MITRE ATT&CK
Credential Access

## Description
Attackers dump credentials from the Local Security Authority Subsystem Service (LSASS) process to obtain passwords, NTLM hashes, and authentication material.
