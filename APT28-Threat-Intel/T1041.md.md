## Technique Name  
Exfiltration Over C2 Channel  
  
## MITRE ATT&CK  
Exfiltration  
  
## Description  
Attackers exfiltrate stolen data through an existing Command and Control (C2) channel to avoid detection.  
  
## Used By  
[[APT28]]  
  
## Data Exfiltrated  
- Credentials  
- Passwords  
- NTLM Hashes  
- Sensitive Documents  
  
## Network Indicators  
- SMTP Port 587  
- SMTPS Port 465  
- TLS Connections  
- Gmail Infrastructure  
  
## Detection  
[[Sysmon]]  
[[Wireshark]]  
  
## Related Events  
- Sysmon Event ID 3 (Network Connection)  
  
## Purpose  
Transfer stolen information to attacker-controlled infrastructure.
